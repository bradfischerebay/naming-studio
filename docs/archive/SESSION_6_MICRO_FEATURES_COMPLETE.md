### 1. Micro-Features Are Substantial

Many "micro-features" are full-fledged programs with:
- Dedicated help documentation
- Multi-year development history
- Complex feature sets (e.g., Product Research: 3 years of data, all eBay marketplaces)
- Significant user base (e.g., Seller Hub free to all sellers, 1M+ users)

**Implication**: "Micro" refers to visibility, not importance. These programs drive seller/buyer experience.

### 2. 2025 Is AI-Focused

5 of 5 programs launched in 2025 use AI:
- AI Assistant for Messaging
- AI Shopping Agent
- Inventory Mapping API (AI-powered)
- Shop the Look (AI-curated)
- Watch List Category Filters (ML-based)

**Implication**: eBay investing heavily in AI-powered seller/buyer tools.

### 3. Discounts Manager Is Umbrella Program

Discounts Manager (2015) encompasses 5 distinct promotional tools:
- Volume Pricing
- Coded Coupons
- Sale Events
- Shipping Discounts
- Order Discounts

**Implication**: Some "programs" are actually feature suites. Need to capture parent-child relationships.

### 4. Seller Hub Is Central Hub

Seller Hub integrates with 20+ programs discovered this session:
- All analytics tools
- All 2025 new features
- Discounts Manager
- Advertising Dashboard

**Implication**: Seller Hub is the primary seller experience. Most new seller features integrate here.

### 5. Legacy vs Current Status Matters

Discovered programs with 3 status types:
- **Current**: 40 programs (93%)
- **Legacy**: 2 programs (Promoted Listings Express, Image Search - disabled 2024)
- **Deprecated**: 1 program (RedLaser standalone app)

**Implication**: Need to track program lifecycle (launch → active → deprecated → sunset).
